Contexte

La problématique

Nous avons trop de tâches et pas assez de temps. Comment choisir lesquelles accomplir pour être le plus productif possible ?

Tâches multiples

Révisions, projets, sport, loisirs les étudiants jonglent avec de nombreuses activités simultanées.

Temps limité

La journée ne fait que 24h. Le temps disponible est une contrainte dure qu'on ne peut pas dépasser.

Priorités difficiles

Quelle tâche choisir en premier ? Comment maximiser le nombre de tâches réalisées ?

Approche algorithmique

En triant les tâches par durée croissante, on peut en accomplir le maximum dans le temps donné.

Question centrale : comment choisir les tâches à effectuer tout en respectant une contrainte de temps et en maximisant le nombre de tâches réalisées ?

Solution

L'algorithme en 5 étapes

on trie les tâches du plus court au plus long, puis on les sélectionne une par une tant que le temps le permet.

1

Saisie des tâches

L'utilisateur entre le temps total disponible, puis le nom et la durée de chaque tâche. Les données sont stockées dans deux tableaux parallèles taches[] et durees[].

2

Tri par durée croissante

On applique un tri à sélection (double boucle) pour réordonner les tâches du plus court au plus long. Les deux tableaux sont synchronisés lors des échanges.

3

Sélection

On parcourt les tâches triées. Si la durée de la tâche courante est inférieure ou égale au temps restant, on l'accepte et on décrémente le temps restant.

4

Marquage des tâches acceptées

Un tableau accepte[] (valeurs 0 ou 1) mémorise quelles tâches ont été sélectionnées, permettant un affichage propre en fin d'algorithme.

5

Affichage du planning

On affiche les tâches acceptées avec leurs durées, ainsi que le temps restant non utilisé. Le résultat est optimal pour maximiser le nombre de tâches accomplies.

Algorithme Gestion_Temps_Optimisee

Variables
    Entier N, i, j
    Entier temps_total, temps_restant
    Chaine taches[50]
    Entier durees[50]
    Entier accepte[50]
    Entier temp
    Chaine tmp

Début
    Lire(temps_total)
    temps_restant ← temps_total
    Lire(N)

    Pour i ← 0 à N-1 pas=1 faire
        Lire(taches[i])
        Lire(durees[i])
        accepte[i] ← 0
    FinPour

    Pour i ← 0 à N-2 pas=1 faire
        Pour j ← i+1 à N-1 pas=1 faire
            Si durees[i] > durees[j] Alors
                temp ← durees[i]
                durees[i] ← durees[j]
                durees[j] ← temp
                tmp ← taches[i]
                taches[i] ← taches[j]
                taches[j] ← tmp
            FinSi
        FinPour
    FinPour

    Pour i ← 0 à N-1 pas=1 faire
        Si durees[i] ≤ temps_restant Alors
            accepte[i] ← 1
            temps_restant ← temps_restant - durees[i]
        FinSi
    FinPour

    Ecrire("Taches executees :")
    Pour i ← 0 à N-1 pas=1 faire
        Si accepte[i] = 1 Alors
            Ecrire(taches[i], " : ", durees[i])
        FinSi
    FinPour
    Ecrire("Temps restant : ", temps_restant)
Fin
#include <stdio.h>
#include <string.h>

int main() {
    int N, i, j;
    int temps_total, temps_restant;
    char taches[50][50];
    int  durees[50];
    int  accepte[50];
    int  temp;
    char tmp[50];

    printf("Donner le temps total : ");
    scanf("%d", &temps_total);
    temps_restant = temps_total;

    printf("Donner le nombre de taches : ");
    scanf("%d", &N);

    for (i = 0; i < N; i++) {
        printf("Nom de la tache %d : ", i+1);
        scanf("%s", taches[i]);
        printf("Duree : ");
        scanf("%d", &durees[i]);
        accepte[i] = 0;
    }

    for (i = 0; i < N-1; i++) {
        for (j = i+1; j < N; j++) {
            if (durees[i] > durees[j]) {
                temp = durees[i];
                durees[i] = durees[j];
                durees[j] = temp;
                strcpy(tmp, taches[i]);
                strcpy(taches[i], taches[j]);
                strcpy(taches[j], tmp);
            }
        }
    }

    for (i = 0; i < N; i++) {
        if (durees[i] <= temps_restant) {
            accepte[i] = 1;
            temps_restant -= durees[i];
        }
    }

    printf("\nTaches executees :\n");
    for (i = 0; i < N; i++) {
        if (accepte[i] == 1)
            printf("  %-20s : %d heure(s)\n", taches[i], durees[i]);
    }
    printf("Temps restant : %d heure(s)\n", temps_restant);

    return 0;
}
